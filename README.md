# 📖 Overview

**PhishGuard** is an interactive cybersecurity awareness platform developed using **Python** and **Streamlit**.

The application helps users understand modern phishing attacks through:

- 📚 Interactive training modules
- 🧠 Knowledge quizzes
- 🎯 Realistic phishing simulations
- 📊 Progress tracking
- 🏆 Achievement badges
- 📜 Personalized certificate generation

The project is designed for **students, employees, and organizations** to improve cybersecurity awareness and reduce phishing-related risks.

---

# ✨ Features

## 📚 Interactive Training

- ✅ 6 cybersecurity learning modules
- ✅ Easy-to-understand explanations
- ✅ Red flags for every phishing attack
- ✅ Best security practices

---

## 🧠 Quiz System

- 🎲 Randomized questions
- 📊 Automatic scoring
- 🔄 Restart quiz anytime
- 🏆 Pass/fail evaluation

---

## 🎯 Phishing Email Simulator

Practice identifying phishing emails by analyzing realistic examples.

Includes:

- 📧 Fake banking emails
- 🎁 Giveaway scams
- 💰 Invoice emails
- 🔍 Detailed explanations after submission

---

## 📊 Progress Dashboard

Track your learning with:

- 📈 Progress bar
- 📉 Interactive charts
- 🏅 Achievement badges
- 📚 Completed modules

---

## 📜 Certificate Generator

After completing the course, users can generate a personalized certificate containing:

- 👤 User name
- 📅 Completion date
- 🏆 Completion status

# ⚙ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/ahmedelharmil30-cyber/CodeAlpha_Phishing-Awareness-Training.git
```

---

## 2️⃣ Navigate to Project

```bash
cd CodeAlpha_Phishing-Awareness-Training
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run Application

